# 简介
vue-element-perfect 是一个后台前端解决方案, 基于Vue3.0+TS+Element-plus实现。它使用了最新的前端技术栈、动态路由，权限验证，并且有着丰富的组件。

## 目录结构介绍

```

├─assets    放置一些静态文件
├─components 放置组件
├─layout 首页布局
│  ├─components
│  │  ├─Sidebar 侧边栏配置
│  │  ├─TagsView 标签栏配置
│  │  └─UHeader 头部内容配置
│  └─hooks
├─router   路由配置
├─store    数据存储
├─styles   样式文件
├─utils   工具方法
└─views    页面配置方法
└─permission  路由拦截 权限配置
```

## 技术栈
vue3.0 + Typescript + vuex + vue-router + Element-Plus scss

## 实现功能
- 登录逻辑
- 权限校验
- 侧边栏导航显示
- 打印
- 地图
- 头像裁剪
- 无限滚动
- 多标签显示
- 退出
- 其他各组件

## 项目 master 分支

```
# 克隆项目
git clone https://github.com/zouzhibin/vue-admin-perfect.git

# 安装依赖
yarn

# 本地开发 启动项目
yarn serve
```
